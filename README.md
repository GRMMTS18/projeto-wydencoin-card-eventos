# WYDENCOIN - Cards de Eventos

## 📌 Descrição
Interface web desenvolvida para exibição de eventos em formato de cards, com foco em organização visual e experiência do usuário.

## 🚀 Funcionalidades
- Exibição de eventos em cards
- Layout responsivo
- Organização visual das informações

## 🛠️ Tecnologias
- HTML
- CSS
- JavaScript

## 💻 Meu papel no projeto
- Desenvolvimento do Front-End
- Criação dos layouts de cards
- Estilização com CSS
- Estruturação das páginas

## 📸 Preview
Adicione prints da interface

## 💡 Objetivo
Projeto acadêmico com foco em prática de desenvolvimento de interfaces web.
